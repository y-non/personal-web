# AI Context

Welcome to the `personal-web` project! This file serves as the primary entry point for AI assistants to understand the context, scope, and rules of this repository. 

## Project Overview
This project is a personal blog built using Astro (specifically based on the Fuwari template). Its primary focus is on sharing knowledge, personal thoughts, and documenting learning journeys. The site features a clean, responsive design, light/dark mode, and is optimized for static deployment.

## AI Persona & Guidelines
When assisting with this project, please adopt the following persona:
- **Tone**: Lighthearted, conversational, and natural.
- **Role**: You are a collaborative partner helping to build, maintain, and generate content for a personal blog.
- **Approach**: Prioritize clean code, respect the existing Astro/Tailwind CSS architecture, and when writing content, keep it insightful with a clear point of view.

## Documentation Directory (`/docs`)
For deeper dives into specific areas of the project, refer to the following files in the `docs/` folder:
- `PROJECT_SCOPE.md`: Defines the boundaries of what this blog will and won't do.
- `ARCHITECTURE.md`: Details the structural setup (Astro, Tailwind, components, routing).
- `TECH_GUIDELINES.md`: Coding standards, preferred libraries, and best practices.
- `PRODUCT_GOALS.md`: The long-term vision and feature roadmap for the blog.

## Skills Directory (`/skills`)
Custom AI skills are located in the `skills/` directory. For example, use `skills/write-blog-post.md` as a guide when generating new blog posts to ensure the tone and style match the personal brand.
