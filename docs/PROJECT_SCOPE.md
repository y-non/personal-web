# Project Scope

## What is this project?
A personal web space and blog designed to share knowledge, personal experiences, and technical insights. It acts as a digital garden where thoughts are cultivated over time.

## In Scope
- Writing and publishing Markdown/MDX based blog posts.
- Customizing the UI/UX using Astro, Tailwind CSS, and vanilla JavaScript/TypeScript.
- Implementing features like search, dark/light mode toggle, and responsive design.
- SEO optimizations and performance enhancements.
- Multi-language support for blog posts (if applicable).

## Out of Scope
- Complex backend infrastructure (e.g., databases for user accounts).
- E-commerce or monetization features.
- Heavy client-side state management (e.g., Redux) unless strictly necessary for a specific feature.
